# Contrast Palette Checker

A small React app for checking whether color pairs and palettes meet WCAG contrast guidelines.

## Features

- Add, edit, copy, and remove palette colors.
- Compare foreground and background colors manually.
- Scan a full palette to see which color pairs pass contrast checks.
- Preview how selected colors look on text, controls, and UI elements.

## Tech

- React
- Vite
- CSS

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
